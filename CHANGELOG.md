## [v0.1.0] - 2026-01-25

### Highlights

- Visual multi-agent workflow designer with React Flow canvas for exploring agent team specifications
- Gmail-style floating panels for viewing agent details with drag, resize, and minimize
- Persistent views to save and restore UI layouts using Ent ORM with SQLite/PostgreSQL

### Added

- React Flow canvas for visualizing agent workflows with custom nodes and edges ([`2987938`](https://github.com/agentplexus/agentplexus/commit/2987938))
- Gmail-style floating panels for agent details with drag, resize, minimize, and close controls ([`2987938`](https://github.com/agentplexus/agentplexus/commit/2987938))
- Multiple floating panels support with staggered positioning (like Gmail compose windows) ([`2987938`](https://github.com/agentplexus/agentplexus/commit/2987938))
- Panel header colors match agent model type (opus=red, sonnet=blue, haiku=green) ([`2987938`](https://github.com/agentplexus/agentplexus/commit/2987938))
- 2x2 grid layout for viewing up to 4 agent teams simultaneously ([`2987938`](https://github.com/agentplexus/agentplexus/commit/2987938))
- Views dropdown to save, load, update, and delete UI layouts ([`2987938`](https://github.com/agentplexus/agentplexus/commit/2987938))
- Markdown rendering with syntax highlighting for agent instructions ([`2987938`](https://github.com/agentplexus/agentplexus/commit/2987938))
- Team selector sidebar with checkboxes for multi-team selection ([`2987938`](https://github.com/agentplexus/agentplexus/commit/2987938))
- Resizable sidebar with drag handle ([`2987938`](https://github.com/agentplexus/agentplexus/commit/2987938))
- Go HTTP server with agent team API endpoints ([`1ae5b0f`](https://github.com/agentplexus/agentplexus/commit/1ae5b0f))
- File watcher for hot-reloading spec changes without server restart ([`1ae5b0f`](https://github.com/agentplexus/agentplexus/commit/1ae5b0f))
- Ent ORM schema for Views entity with SQLite and PostgreSQL support ([`1ae5b0f`](https://github.com/agentplexus/agentplexus/commit/1ae5b0f))
- Views REST API for CRUD operations on saved layouts ([`1ae5b0f`](https://github.com/agentplexus/agentplexus/commit/1ae5b0f))
- Environment-based database configuration (DB_DRIVER, DB_DSN) ([`1ae5b0f`](https://github.com/agentplexus/agentplexus/commit/1ae5b0f))
- CLI entry point with workspace and port flags ([`1ae5b0f`](https://github.com/agentplexus/agentplexus/commit/1ae5b0f))
- Embedded frontend assets in Go binary ([`1ae5b0f`](https://github.com/agentplexus/agentplexus/commit/1ae5b0f))
